# Jaehee’s Garden

게임과 책, 아직 자라는 생각을 기록하는 개인 정원입니다.

## Stack

- Astro 7 + MDX
- Astro Content Collections
- Cloudflare Pages

## Local development

```bash
npm install
npm run dev
```

## Cloudflare Pages

- Build command: `npm run build`
- Build output directory: `dist`
- Node.js: 22 or later

글은 `src/content/writing`, 책은 `src/content/books`에 추가합니다.
